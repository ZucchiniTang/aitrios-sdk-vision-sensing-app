# Convert Annotation VoTT Format to COCO Format

With the notebook, you can convert an annotation file tagged and exported with VoTT into COCO format.

## Getting Started
You need an VoTT format annotation file to start this tutorial. Tag images using VoTT and get the annotation file. See [the document](../tag_images_with_vott/tag_images_with_vott.md) for details.

### Start workflow
Follow the instructions in the [notebook](./convert_annotation_from_vott_to_coco.ipynb) to convert the annotation file format.

>**NOTE**
>
>You can run the cells one by one. (You can NOT run all cells at once.)
